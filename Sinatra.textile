You can use Cucumber with "Sinatra":http://github.com/bmizerany/sinatra and "Webrat":http://www.brynary.com/uploads/webrat/rdoc/index.html!

Just make sure you have a <code>features/support/env.rb</code> file with the following:

<pre><code>
# RSpec matchers
require 'spec/expectations'

# Webrat
require 'webrat'
Webrat.configure do |config|
  config.mode = :sinatra
end

World do
  Webrat::SinatraSession.new
end

# Sinatra - my views are not in the cucumber bin dir!
$0 = File.dirname(__FILE__) + '/../../your_sinatra_app.rb'
# Note: The above didn't work for me, I do:
# set :views, File.join(File.dirname(__FILE__), 'views') in my sinatra app instead, just to be sure.

require File.dirname(__FILE__) + '/../../your_sinatra_app'
</code></pre>
