h2. All the 'a' and 'A' letters are lost in the output on Windows

If <code>$KCODE='u'</code> is set by your code or some of the libraries you are using, you will lose _a_ and _A_ in the console output.

Solution: Set <code>$KCODE=''</code> in your <code>env.rb</code> file. For more info, see "Lighthouse issue #81":http://rspec.lighthouseapp.com/projects/16211/tickets/81-windows-all-the-a-characters-in-the-output-have-gone-on-strike
