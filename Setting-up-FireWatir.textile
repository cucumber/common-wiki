h2. Step 1: Install the Watir gem

<pre><code>gem install watir </code></pre>

h2. Step 2: Install jssh.xpi

Install the jssh.xpi relating to your OS and Firefox version:

http://wiki.openqa.org/display/WTR/FireWatir+Installation

h2. Step 3: Start Firefox with the jssh switch

This is a one off. 

From the command line goto your Mozilla Firefox installation folder and type:

<pre><code>firefox -jssh </code></pre>

h2. Step 4: Test your protocol

From the command line type:

<pre><code>telnet localhost 9997</code></pre>

This should return 'Welcome to the Mozilla JavaScript Shell!'
